Aplikasi Perpustakaan kami memiliki 2 role, yaitu admmin dan user. Dalam halaman user akan ditampilkan deskripsi singkat mengenai perpustakaan 
dan juga ada tabel peminjaman, 
yang mana peminjaman itu nanti jika ditambahkan akan masuk kedalam data admin.
Pada halaman admin berisi data buku, data peminjam, dan riwayat.
Library yang kami gunakan untuk mengembangkan website kami 
adalah bootstrap, java script, php, php my admin, html dan laravel.
